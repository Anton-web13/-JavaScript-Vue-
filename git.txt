Вывести коммит с именем, текстом и т.д:
git log --pretty=format:'%h %an: %s'
git log --pretty=format:'%Cgreen %h %Creset %an: %s'
git log --pretty=format:'%Cred %h %Creset %an: %s'
git log --pretty=format:'%Cred %h %Cgreen %an: %Creset %s'
git log --pretty=format:'[%h] %Cgreen %an: %Creset %s'
git log --pretty=format:'%C(yellow)[%h] %Cgreen %an: %Creset %s'
---------------------------------------------------------------------------------------------------
Создание своих команд:
git config --global alias.logp "log --pretty=format:'[%h] %Cgreen %an: %Creset %s'"
